# OctoBot Labs Service Catalog

This catalog defines the first sellable OctoBot Labs offers. The delivery stance is technical and compliance-adjacent: OctoBot Labs provides workflow review, evidence, implementation support, and practical risk reduction. It does not provide legal advice, certification, penetration testing, or a security guarantee.

## 1. AI Workflow Mini-Audit

**Buyer:** founder-led SMEs, AI automation agencies, small SaaS teams, and operators using n8n, Make, Zapier, MCP servers, internal LLM workflows, or AI agents with real tool access.

**Price:** EUR 750-1,200 fixed fee.

**Scope:** one workflow, one agent surface, one MCP server surface, or one small automation path.

**Inputs:**

- Sanitized workflow export, screenshots, or screen-share walkthrough
- Tool list and permission summary
- Description of read/write actions
- Known concerns from the owner

**Deliverables:**

- 5-8 page mini-audit report
- Risk snapshot across credentials, permissions, human approval, logging, data handling, and recovery
- 3-7 prioritized findings
- Fix-now and fix-later backlog
- Optional next-step recommendation

**Timeline:** 5 business days after complete materials.

**AI automation target:** 55-70 percent for inventory extraction, checklist coverage, report drafting, and remediation-ticket drafts. Human review owns severity, business impact, and final wording.

## 2. AI Workflow Security Audit

**Buyer:** SMEs and B2B teams running multiple AI workflows or connecting agents to business-critical systems.

**Price:** EUR 2,000-4,500 depending on workflow count, system access, and reporting depth.

**Scope:** two to six workflows, or one connected workflow family.

**Deliverables:**

- Workflow inventory
- Credential and OAuth/service-account review
- Permission map
- Human approval and autonomy review
- Audit trail and logging review
- Data handling map
- Reliability and recovery review
- Ranked findings with evidence and remediation steps
- AI workflow evidence pack for internal review

**Timeline:** 7-15 business days after access and materials.

**AI automation target:** 55-70 percent for inventory, evidence organization, risk-map drafts, and report sections. Human review owns prioritization and security-sensitive decisions.

## 3. AI Fix Sprint

**Buyer:** audit clients that want the highest-priority fixes implemented.

**Price:** EUR 3,500-9,000 depending on scope.

**Scope examples:**

- Move API keys and tokens into safer secret storage
- Reduce OAuth/API scopes
- Add human approval gates before write actions
- Add logging for tool calls, approvals, errors, and config changes
- Add duplicate-action prevention and recovery steps
- Improve MCP server/tool authorization boundaries

**Deliverables:**

- Agreed implementation backlog
- Implemented fixes in the client environment or pull-request-ready changes
- Short change log
- Updated workflow diagram or runbook
- Residual-risk note

**Timeline:** usually 1-3 weeks.

**AI automation target:** 45-65 percent for implementation drafts, tests, runbooks, and ticket writing. Human review owns credentials, permissions, production changes, and final validation.

## 4. Monitoring Retainer

**Buyer:** clients with ongoing workflow changes or growing AI usage.

**Price:** EUR 750-1,500 per month.

**Scope:**

- Monthly workflow change review
- Permission drift check
- Review of new tools, models, providers, and MCP servers
- Lightweight incident/risk log
- Async implementation questions within agreed limits
- Monthly findings summary

**Exclusions:**

- Unlimited implementation
- 24/7 incident response
- Legal opinions
- Compliance certification
- Full penetration testing

**AI automation target:** 70-85 percent for monitoring summaries, change-log drafting, recurring checklist runs, and reminder workflows. Human review owns final judgement and client-facing recommendations.
