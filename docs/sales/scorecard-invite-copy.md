# Scorecard Invite Copy

Date: 2026-04-26

Use this from `hello@octo-bot.io`. Keep the tone low-friction and research-oriented. The first ask is not a paid audit. The first ask is: try the scorecard on one real workflow and tell us whether it catches the right risks.

## Sending Principles

- Use company/team-level personalization, not personal scraping.
- Reference one public signal from the prospect's website or partner profile.
- Ask for a 3-minute scorecard test, not a meeting.
- Tell them the scorecard runs in the browser and that they should not enter secrets.
- Offer a small useful exchange: if they send the result, OctoBot Labs replies with the first 1-2 fixes we would prioritize.
- For German prospects, use official partner-directory contact buttons or public company contact forms first.
