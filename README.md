# octo-bot

OctoBot contains two active tracks:

- `octo-bot-app/` - the existing Expo/Supabase app for tracking business obligations from email.
- `site/` - the static OctoBot Labs offer site for AI workflow security audits.

The app direction is preserved. The public site is intentionally separate so the higher-ticket service offer can be tested without disturbing product development.
