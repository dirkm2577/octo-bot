# OctoBot Labs Static Site

This folder contains the first public offer site for OctoBot Labs.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static file server.

## Deployment

Deploy the contents of this folder as the public web root for `octo-bot.io`.

Before launch, configure a real inbox for:

```text
hello@octo-bot.io
```

The current CTA links use `mailto:` so the site can launch without a backend form.
